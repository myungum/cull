수확의 낫이 과연 승리를 가져다줄 수 있을까요?

## 방법
1. 소환사명 : 프로문의는DM으로
2. 최근 930 게임의 승패를 분석합니다.
3. 클래식 게임만 인정합니다.
4. 다시하기는 제외합니다.
> 다시하기 : 180초 이내 특정 조건 만족 시 게임을 무효로 할 수 있습니다.

## 결과
구분 | 승 | 패 | 총 게임 | 승률
---- | ---- | ---- | ---- | ----
수확의 낫을 산 경우 | 106 | 104 | 210 | 50.4%
수확의 낫을 사지 않은 경우 | 380 | 340 | 720 | 52.7%

위 표와 같이 수확의 낫을 <b>사지 않았을 경우</b> 더 높은 승률을 보여주었습니다.<br>
